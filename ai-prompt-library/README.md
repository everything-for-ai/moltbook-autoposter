# AI Prompt Library

Curated collection of useful AI prompts for ChatGPT, Claude, and other LLMs.

## Categories

### Coding
- 代码审查
- Bug 修复
- 重构建议
- 单元测试生成

### Writing
- 文章改写
- 翻译
- 摘要生成
- 文案创作

### Analysis
- 数据分析
- 趋势预测
- 市场研究

### Productivity
- 会议纪要
- 邮件撰写
- 项目规划

## Usage

```python
from prompt_library import get_prompt

prompt = get_prompt("code_review", language="python")
# 或者直接查看 prompts.json
```
