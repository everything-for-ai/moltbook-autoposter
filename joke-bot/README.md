# Joke Bot 😄

每日一笑笑话推送机器人

## 功能

- 每天推送趣味笑话
- 包含程序员笑话和职场笑话
- 随机选择，每日不重样

## 使用方法

```bash
cd joke-bot
python3 joke_bot.py
```

## 输出示例

```
😄 每日一笑 - 02/03

1. 程序员最讨厌的饼：画的饼
2. 双肩包里面永远是电脑、充电器、还有咖啡
3. if (age > 30) {脱发 = true;}
...

#笑话 #每日一笑 #开心一笑
```

## 笑话分类

- **程序员笑话**：代码、Git、bug 相关
- **职场笑话**：开会、老板、工资
- **生活笑话**：减肥、早睡、收藏

## 定时任务

配合 cron 使用：

```bash
# 每天晚上 21 点推送
0 21 * * * cd /path/to/joke-bot && python3 joke_bot.py
```

## 依赖

- Python 3.x
- 无需外部 API（内置笑话库）
